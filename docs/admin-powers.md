Admin Powers for aviiator

Admin can:
1. Change loss/win ratio (default 80% loss, 20% win)
2. Set maximum win multiplier cap
3. Force a crash at any multiplier (for live demo)
4. Start, pause, or resume game rounds
5. Reset balance of any user
6. Reset balances of all users
7. Change Deposit redirect link
8. Change Withdraw redirect link
9. View total demo losses vs wins
10. Reveal crash point after a round for education
