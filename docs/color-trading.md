Color Trading Module for aviiator

Purpose:
Educational demo to show why color trading leads to losses.

Rules:
- Demo credits only
- No real money
- Colors: Red, Green, Violet
- Each round has a fixed timer
- Server decides winning color before round starts
- User selects one color per round

Bias:
- Majority rounds result in user loss
- Winning payouts are capped
- Over time, total wins never exceed losses

Flow:
1. User selects color
2. Timer runs
3. Result is revealed
4. User mostly loses
5. Next round starts automatically

Admin Controls:
- Enable/disable color trading
- Control loss bias
- Force result for demo
