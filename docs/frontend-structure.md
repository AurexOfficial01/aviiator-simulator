Frontend Structure for aviiator

frontend/
 ├─ index.html        -> main page
 ├─ game/
 │   ├─ game.html     -> plane + multiplier screen
 │   └─ controls.js   -> play & cashout buttons
 ├─ auth/
 │   ├─ login.html    -> user login
 │   └─ signup.html   -> user signup
 ├─ admin/
 │   └─ admin.html    -> admin panel
 ├─ lang/
 │   ├─ en.json       -> English text
 │   └─ hi.json       -> Hindi text
 └─ assets/
     └─ plane.png     -> plane image
