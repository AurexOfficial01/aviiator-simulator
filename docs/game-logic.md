Game Logic for aviiator

1. A round starts automatically.
2. Plane appears at 1.00x.
3. Multiplier increases smoothly with time.
4. Server has already decided the crash point.
5. User can press CASHOUT anytime before crash.
6. If user cashes out before crash:
   - User wins demo credits.
7. If plane crashes before cashout:
   - User loses demo credits.
8. 80% of rounds are designed to cause loss.
9. 20% of rounds allow win but with capped multiplier.
10. Over long play, total user wins never exceed losses.
11. Deposit and Withdraw buttons do not affect balance.
12. Clicking Deposit or Withdraw redirects user to a page defined by admin.
13. Game never stops; rounds repeat automatically.
