Platform Name: aviiator

Purpose:
Educational crash-game simulator for awareness.

Money:
No real money.
Only demo credits.

Game Rules:
- Plane starts at 1.00x
- Multiplier increases smoothly
- Crash point is decided by server before round
- 80% rounds are loss
- 20% rounds are win
- Wins are capped so total wins never exceed total losses
- Platform always remains net positive

Buttons:
Deposit and Withdraw buttons are visible.
On click, they redirect to admin-defined links.
No money action happens.

Users:
- Multiple users at same time
- Unlimited play
- Demo balance only

Languages:
English and Hindi with toggle.

Admin:
Admin can control bias, crash, balances, and links.
