User Flow for aviiator

1. User opens the website.
2. User signs up or logs in.
3. User sees demo balance.
4. User enters a demo amount to play.
5. Plane round starts automatically.
6. User watches multiplier increase.
7. User tries to cash out.
8. In most cases, user loses.
9. Sometimes user wins but gains are limited.
10. Balance decreases over time.
11. User can play again immediately.
12. User can click Deposit or Withdraw and gets redirected.
13. User can play unlimited rounds.
